# 📚 Study Planner

A modern, intuitive, and efficient study session management application built with **React 19** and **Tailwind CSS 4**.

![Study Planner Banner](https://img.shields.io/badge/Study--Planner-React-61DAFB?style=for-the-badge&logo=react)
![Tailwind CSS](https://img.shields.io/badge/Tailwind--CSS-v4-38B2AC?style=for-the-badge&logo=tailwind-css)
![Vite](https://img.shields.io/badge/Vite-v7-646CFF?style=for-the-badge&logo=vite)

## ✨ Features

- **🚀 Session Management**: Effortlessly create, edit, and delete study sessions.
- **🕒 Real-time Updates**: Immediate feedback and state management across the app.
- **📱 Responsive Design**: Fully optimized for various devices using Tailwind CSS 4.
- **🛠️ Robust Forms**: Built with `react-hook-form` for seamless user input handling.
- **🆔 Unique Identifiers**: Uses `nanoid` for reliable key generation.
- **💡 State Management**: Utilizes React Context API for centralized data handling.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Forms**: [React Hook Form](https://react-hook-form.com/)
- **Utilities**: [nanoid](https://github.com/ai/nanoid)

## 🚀 Getting Started

Follow these steps to set up the project locally:

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/S-lakhera/Kodex-react.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Study-Planner
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

### Running the App

Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.
